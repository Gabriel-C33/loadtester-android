# Kivy Launcher - Работещ метод БЕЗ build!

## Стъпки:

### 1. Инсталирай Kivy Launcher на телефона
- Отвори Google Play Store
- Търси "Kivy Launcher"
- Инсталирай: https://play.google.com/store/apps/details?id=org.kivy.pygame

### 2. Създай папка на телефона
Създай тази структура на телефона:
```
/sdcard/kivy/loadtester/
```

### 3. Копирай файловете
Копирай тези файлове в `/sdcard/kivy/loadtester/`:
- `main.py`
- `locustfile.py`
- `android.txt` (ще го създам сега)

### 4. Стартирай
- Отвори Kivy Launcher
- Избери "loadtester"
- Готово!
